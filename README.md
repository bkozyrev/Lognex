#Тестовое задание для соискателей на позицию разработчика-стажёра на Java

Инструкции по запуску:

Вариант 1:

 * В командной строке перейти в корневой каталог проекта
 * Выполнить gradlew run
 * В появившемся окне выбрать исходный txt файл и нажать 'Ok'.
 * Циклические зависимости выводятся в STDOUT.

Вариант 2:

 * В командной строке перейти в каталог ..\Lognex\build\libs
 * Выполнить java -jar Lognex-1.0-SNAPSHOT.jar
 
Вариант 3:
 
 * Импортировать файл build.gradle в среду разработки.
 * Перейти в класс TestTask и выполнить Run TestTask.main().
